# Move Zeros to End

Use 2 pointers, replace 0 with non-zero element

- Use 2 pointers $(i,j)$
- Move first pointer to 0, second pointer to element
- Swap both, and increment first pointer

```
void moveZeros(int arr[], int n)
  int count = 0;
  for (int i = 0; i < n; i++;)
    if (arr[i] != 0)
      swap(arr[i], arr[count])
      count++;
```
