# Left Rotate an Array by D places

- Reverse $(0,d-1)$
- Reverse $(d, n-1)$
- Reverse $(0, n-1)$

```
void lRotateByD(int arr[], int n, int d)
  reverse(arr, 0, d-1)
  reverse(arr, d, n-1)
  reverse(arr, 0, n-1)
```

# Left Rotate an Array by 1

Store first element in temp, move all elements towards left, put temp in last

```
void lRotateOne(int arr[], int n)
  int temp = arr[0]
  for (i=1; i<n; i++;)
    arr[i-1] = arr[i]
  arr[n-1] = temp
```
