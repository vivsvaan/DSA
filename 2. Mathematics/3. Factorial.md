# Factorial

### Iterative

- Set result = 1
- Multiple numbers from 2 till $n$ and add in result

```
int factorial(int n)
  int res = 1
  for (int i=2; i<=n; i++;)
    res = res * i

  return res
```

### Recursion

- call $n*factorial(n-1)$

```
int factorial(int n)
  if (n == 0)
    return 1;
  return n*factorial(n-1)
```
